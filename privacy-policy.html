Privacy Policy

Last Updated: June 2026

Haptik Action Downloader is a Chrome Extension designed to allow authorized users to download action code from Haptik AI Agents.

Information Collected

  • Google account email address
  • Authentication status stored locally in Chrome Storage

How Information Is Used

The email address is used solely to verify that the user belongs to an authorized Haptik domain and to control access to the extension.

Data Storage

Authentication information is stored locally in the user's browser using Chrome Storage.

Data Sharing

No personal information is sold, shared, rented, or transferred to third parties.

Security

Authentication is performed using Google OAuth services.
